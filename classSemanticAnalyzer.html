Detailed Description

Analyzes the semantics of a given set of lexems.

The SemanticAnalyzer class is responsible for performing semantic analysis on a vector of lexems. It checks for correct usage of variables, functions, and other language constructs within the provided lexems.

Prints the function signatures stored in the SemanticAnalyzer.
