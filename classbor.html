Adds a string to the trie data structure.

This function traverses the trie character by character, creating new vertices as needed, and marks the last vertex as terminal.

Parameters
sThe string to be added to the trie

Time Complexity: O(|s|) where |s| is the length of the input string Space Complexity: O(|s|) in worst case when creating new vertices

Checks if a given string exists in the trie and returns the corresponding lexeme.

Parameters
sPointer to the character array to check
sizeLength of the string to check
s_indexStarting index of the string in the source code
Returns
std::pair<bool, Lexem> First element indicates if string was found, second element contains the constructed Lexem regardless of whether string was found

The function traverses the trie character by character checking if the given string exists. If at any point a character is not found in the trie, returns false with a partial lexeme. If the entire string is found and ends at a terminal node, returns true with the complete lexeme. Otherwise returns false with the complete but non-terminal lexeme.
